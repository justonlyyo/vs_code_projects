
# 接口定义

* 通讯录
* 手机识别码
* 经纬度

## 通讯录

* url: /contacts
* method: post
* request type: json
* request args: 

  |参数名| 含义|类型|说明|
  |:-:|:-:|:-:|:-:|
  |contacts|联系人|json序列化的数组对象|手机通讯录内容|
  |device|手机信息|json序列化的字典对象|手机设备信息|
  |registration_id|注册id|字符串|极光的id,有唯一性|
* response type: json

## 经纬度信息

* url: /location
* method: post
* request type: json
* request args: 

  |参数名| 含义|类型|说明|
  |:-:|:-:|:-:|:-:|
  |longitude|经度|经度|float|
  |latitude|纬度|纬度|float|
* response type: json
